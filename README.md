# Building-an-AI-Classifier-Identifying-Cats-Dogs-Pandas-with-PyTorch

## Project Overview
This project builds an AI-based image classifier to identify cats, dogs, and pandas using transfer learning with VGG19 in PyTorch.
The model is trained on a labeled dataset and learns to classify images into three categories with good accuracy.

## Features
- Uses VGG19 pre-trained model
- Applies transfer learning
- Classifies images into: Cat, Dog and Panda
- Built using PyTorch
- Includes training and testing pipeline

## Technologies Used
- Python
- PyTorch
- Torchvision
- NumPy
- Matplotlib

## How It Works
- Load and preprocess image dataset
- Use VGG19 model with modified final layer
- Train the model on training data
- Evaluate performance on test data
- Predict class labels for new images

## Result
The model successfully classifies images of cats, dogs, and pandas with good accuracy using transfer learning.

## Conclusion
This project demonstrates how pre-trained deep learning models like VGG19 can be used for image classification tasks efficiently with limited data.
